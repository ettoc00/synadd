# SynAdd

A powerful, high-performance additive synthesizer built with React, TypeScript, and the Web Audio API.

## ğŸš€ Features

- **Harmonic Control**: Precise adjustment of up to 32 harmonics with intuitive slider controls.
- **Waveform Visualization**: Real-time visualization of the resulting waveform and individual harmonics.
- **Draw Mode**: Intuitive mouse-based drawing to sculpt complex harmonic spectra directly on the canvas.
- **Polyphony**: Fully polyphonic synth engine with a dynamic voice pool for smooth, overlapping notes.
- **Presets**: Built-in preset management to save and load complex harmonic configurations.
- **Master Effects**: Global controls for gain and total harmonic count.
- **Keyboard Interaction**: Play the synthesizer directly from your computer keyboard.
